碩士修業暨研究規劃

北科大 AI 科技碩士學位學程 (C73)

研究題目
整合 LLM 斡旋與數位孿生驗證之原子化界面設計
Y1 上

基礎建置：資料探勘與模型架構

  • ● 深度學習 (3)：掌握 Transformer 與 Attention 機制，建立 LLM 基石。
  • ● 大數據技術與管理 (3)：處理 MES Log 清洗 (ETL)，建置 RAG 知識庫。
  • ● 專題討論 (1)：確立「原子化界面」設計規範與文獻回顧。
Y1 下

原型開發：數位孿生與介面

  • ● 物聯網與感測網路 (3)：串接 Sensor 數據，打造數位孿生驗證沙盒。
  • ● 社群媒體與對話機器人 (3)：開發 LLM Agent，供技術員查詢 SOP。
  • ● 資料可視化 (3)：實作原子化界面，設計主管決策儀表板。
暑期

系統整合測試

目標：串接 NLP 模組與 IoT 數據流，模擬邊界案例進行壓力測試，並完成論文 Proposal 撰寫。

Y2 上

場域實證：演算法優化

  • ● 自然語言處理 (3)：Fine-tuning 模型以適應半導體專有名詞。
  • ● 深度學習應用開發實務 (3)：將異常偵測模型容器化並部署至邊緣端。
  • ● 碩士論文 (3)：於產線進行 A/B Test，收集實驗數據。
Y2 下

成果驗收：系統總整

  • ● 大數據與人工智能應用系統設計 (3)：完成全系統 (界面-LLM-資料庫) 整合。
  • ● 碩士論文 (3)：論證管理失效之改善成效，完成學位口試。

Core Value

本計畫透過系統性修課，構建具備「數位孿生驗證機制的 LLM 斡旋系統」，解決半導體產線之管理失效問題。
