Y1

碩一上：技術底層與場域理解

重點：掌握 GenAI 核心與半導體品管流程

  • 核心修課：修習《生成式人工智慧與應用》，建立 RAG (檢索增強生成) 技術基礎，解決 SOP 檢索問題。
  • 場域知識：修習《品質管制觀念與方法》，理解機台誤差與統計製程管制 (SPC)，為數位孿生的參數設定做準備。
  • 資料準備：對接實習單位 MES 數據，進行初步的資料清洗與向量化 (Vectorization) 測試。
產學創新專題講座 生成式 AI 品質管制
Y1

碩一下：人機協作與預測模型

重點：建構「斡旋者」邏輯與異常偵測模型

  • 核心修課：修習《強化學習於人機互動與機器人》，設計當 LLM 與人類指令衝突時的「權限仲裁」演算法。
  • 數據分析：修習《時間序列分析與預測》，利用機台 Sensor Log 訓練異常偵測模型，作為數位孿生的判斷依據。
  • 介面原型：修習《多媒體互動系統設計與實作》，開發「原子化界面」的原型。
產學創新專題講座 RL 於人機互動 時間序列分析

暑期：系統整合與沙盒驗證

重點：串接 LLM 轉譯器與數位孿生沙盒

  • 系統串接：將 RAG 知識庫 (SOP) 與 時間序列模型整合，實作「指令預演沙盒」。
  • 驗證測試：輸入歷史異常指令，測試系統是否能成功攔截並生成正確的警告 Prompt。
  • 論文計畫：完成論文 Proposal，定義實驗組（使用系統）與對照組（傳統操作）的評估指標。
Y2

碩二上：視覺化優化與落地實驗

重點：數位孿生可視化與場域實驗

  • 視覺化修課：選修《虛擬實境實務開發》或 WebGL 相關課程，優化數位孿生在主管端的 3D/2D 監控儀表板。
  • 演算法優化：修習《最佳化理論與應用》，優化派工與參數調整的建議邏輯。
  • 場域實驗：於實習場域進行小規模導入，收集技術員的操作時間 (Time to complete) 與錯誤率數據。
虛擬實境開發 最佳化理論
Y2

碩二下：論文產出

重點：驗證「管理失效」的改善成效

  • 撰寫碩士論文：整合大型語言模型斡旋與數位孿生驗證之原子化界面設計。
  • 數據驗證：透過 A/B Test 證明系統能有效降低人為疏失與 Downtime。
  • 完成學位口試。

研究核心目標

痛點解決：透過 LLM 解決半導體產線「管理失效」與溝通斷層問題。

技術整合：利用「生成式 AI」做翻譯，「數位孿生」做沙盒，「原子化界面」做呈現，打造具安全驗證機制的智慧人機協作系統。
