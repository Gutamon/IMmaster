輔仁大學資訊管理學系 碩士班讀書計畫

研究主題：自然語言驅動機台智慧運作

依據 114 學年度必選修科目表規劃
Y1

碩一上 (Fall)

重點：基礎建構與題目確立

  • 修習必修「研究方法」(3學分)：確立實驗設計與論文架構
  • 修習必修「資訊管理專題研討」(1學分)：與專家交流找尋靈感
  • 選修「人工智慧」(3學分)：(研究重點：Transformer 模型與 NLP 基礎)
  • 選修「資料庫管理」(3學分)：(研究重點：機台狀態資料的儲存與檢索)
Y1

碩一下 (Spring)

重點：系統架構與原型實作

  • 修習必修「資訊管理專題研討」(1學分)
  • 選修「資料探勘」(3學分)：分析機台 Log 與異常偵測
  • 選修「軟體工程」或「系統分析與設計」：設計 LLM Agent 架構
  • 完成論文計畫書 (Proposal)：確認「自然語言介面」之可行性
暑

升碩二 (Summer)

重點：系統深造與模擬驗證

  • 獨立研究：整合 LLM API 與機台模擬器 (Digital Twin)
  • 解決 Hallucination：實作 RAG (檢索增強生成)
  • 收集實驗數據：指令轉換準確率與回應時間
  • 投稿相關研討會 (如 ICIM) 獲取學術回饋
Y2

碩二上 (Fall)

重點：實驗評估與論文撰寫

  • 修習必修「資訊管理專題研討」(1學分)
  • 進行使用者測試 (User Study)：比較語音/文字指令與傳統介面之差異
  • 開始撰寫學位論文：完成緒論、方法論與系統實作章節
  • 確認滿足畢業門檻：英文檢定 (TOEIC 750) 與 學術倫理測驗
Y2

碩二下 (Spring)

重點：學位口試與畢業

  • 修習必修「資訊管理專題研討」(1學分)
  • 完成論文結論與未來展望 (Future Work)
  • 碩士學位論文口試 (Thesis Defense)
